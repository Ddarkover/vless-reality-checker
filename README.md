# VLESS+Reality — Camouflage Site Checker

Консольная утилита для проверки веб-сайтов на пригодность в качестве камуфляжного адреса для **VLESS+Reality**.

---

## Что проверяется

| # | Критерий |
|---|----------|
| 1 | Сервер находится **вне РФ** (GeoIP) |
| 2 | Поддержка **TLS 1.3** и **HTTP/2** |
| 3 | Главная страница **не редиректит** на другой домен |
| 4 | Географическое **расстояние** между сайтом и вашим сервером |
| 5 | Сообщения после **Server Hello шифруются** (гарантировано TLS 1.3 / RFC 8446) |
| 6 | Поддержка **OCSP Stapling** |

---

## Скачать готовый EXE (Windows)

Перейди в раздел [**Releases**](../../releases) и скачай `checker.exe` из последней версии.  
Python **не требуется**.

---

## Запуск

### Готовый EXE (рекомендуется для Windows)

```
checker.exe
```

Или передай домены сразу в аргументах:

```
checker.exe google.com github.com
```

### Из исходников (Python 3.10+)

```bash
pip install -r requirements.txt
python checker.py
```

## Сборка EXE вручную

```bash
pip install pyinstaller
pyinstaller --onefile --name checker --console checker.py
# Результат: dist/checker.exe
```

Автоматическая сборка настроена через **GitHub Actions** — при создании тега `v*` EXE появляется в Releases автоматически.
